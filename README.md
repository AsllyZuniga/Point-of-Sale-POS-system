## Description

This project is a **modern Point of Sale (POS) platform** developed with **NestJS** and **TypeORM** on the backend, and **Next.js** on the frontend.

It is designed as a real-world application to demonstrate clean architecture, modular design, and scalable full-stack development using one of the most in-demand technology stacks in the industry.

This repository is suitable for **professional portfolios** and showcases practical experience with enterprise-level backend and frontend integration.

---

## Tech Stack

- **Backend:** NestJS, TypeORM, TypeScript
- **Frontend:** Next.js, React
- **Database:** PostgreSQL / MySQL (configurable)
- **API:** REST
- **Authentication:** JWT (planned / optional)
